# 产品及行业层面
这个你更懂，我只说下自己见解
+ 当前网站选品方式：按流行最新产品选品迭代方式进行的；可能达不到扩充效果，不能扩充，风险就比较大，比如某款产品GG了，网站上架的配件会跟着GG。
+ 扩充产品及下架原则：
  + 按流行最新产品进行扩充
  + 按经典产品进行扩充
  + 按市场营收进行保留
  + 客户信息分析（职业、购买配件目的、地域等），以扩充必要的产品，比如手机配件+螺丝刀，螺丝刀是扩充的产品
  + 产品流行度，按地域进行分析
